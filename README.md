# PolygonEditor
Требуется отобразить изображение и обвести на нем некоторые объекты, например здания.

Задачи:
    • Загрузить изображение;
    • Отобразить изображение в виджете;
    • Реализовать нанесение нескольких объектов (полигонов) мышью;
    • Реализовать перемещение выбранного полигона с помощью мыши;
    • Реализовать простое редактирование нарисованного полигона путем перетаскивания вершин мышью.

Язык программирования c++, стандарт 11/14/17;
Для GUI использовать библиотеку Qt5 (Widgets);
Система сборки cmake;
